WBA OpenRoaming — Hackathon Test Network

Secure OpenRoaming network available for the Hackathon, enabling seamless and automatic handover between Cellular ⇄ Wi-Fi for cross-domain experiments (RADIUS, L4S, SCONE, RAW, DetNet, etc.).

Quick facts

  • SSID (test): OpenRoaming_IETF_Montreal
  • Test realm / RADIUS: partners.orportal.org
  • Example assets: RADIUS servers & accounting, PKI APIs, Identity servers, QUIC/HTTP servers, etc.

How to get a profile

⬇ Download OpenRoaming‑Passpoint Profile

Install & test

  • Attach via OpenRoaming‑Passpoint (device should start scanning automatically).
  • (Observational) Open a QUIC HTTP session (port 4443).
  • Trigger handover between Cellular/Wi‑Fi and observe session continuity.
  • Run throughput and latency tests; capture RADIUS packets / accounting.
